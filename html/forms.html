Forms

Forms have many uses.

  • Login/Registrations details
  • Customer feedback
  • Live chat
  • Bookings and purchases
  • Oppinion Polls
  • Any information that needs a user to submit

They allow input of data from the user. Essential to interaction between the user and the site. Contains other elements to make up the form.
They may be text input fields (single or multiple line), e-mail fields, passwords, select boxes, radio buttons, submit and reset etc.
