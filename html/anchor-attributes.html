Download

Specifies that the target will be downloaded when a user clicks on the hyperlink.

The option value of the download attribute will be the new name of the file that is created. If the value is omitted, the original file name is used. See second download link

eg. File Download

<a href"../assets/css/images/pokeball.png" download="example">File Download</a>

Look at the original file name in assets/css/images and the file name given when downloading.


<a href"../assets/css/images/pokeball.png" download="">File Download</a>

eg. File Download


The filename is the same as the original that is being downloaded from.
