Download

Specifies that the target will be downloaded when a user clicks on the hyperlink.

The optional value of the download attribute will be the new name of the file that is created. If the value is omitted, the original file name is used - See second download link

Href

Provides the URL [Uniform Resource Locator] of the linked document.

Href Lang

Specifies the language of the linked resource.

Media

Specifies what device or media format the linked doc is optimized for. Attribute is purely advisory.

Ping

Ping attribute sends a short HTTP POST request to the specified URL. Can be used for monitoring/tracking.
