Id

Attribute

Multiple ids can be applied to one element.
Ids must be unique, they can only apply to one element.
Ids must not have spaces in.

Single id applied to an element:

<div id="example1">

Multiple ids on one element:

Note: all ids sit inside "quote" marks and are separated from each other with a space.

<div id="example1 example2 example-three">

Target in css using #id-name
