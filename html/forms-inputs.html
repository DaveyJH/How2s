Form Inputs

Form inputs are fields for inputting data. Several attributes which must be set for correct data management.

<label for="one">First Example: </label>
<input type="text" id="one" name="this">
<label for="two">Second Example: </label>
<input type="email" id="two" name="this">
<input type="submit"> <input type="reset">

The label for must match the id of an input.
The information from the above form would result in "one":"Data-one", "two":"Data-two"
