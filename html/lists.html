Lists

Ordered Lists

  1. Items in a list.
  2. Items will be numbered.
  3. Used for things which must stay in order.
  4. The steps of making a recipe for example.

Ordered lists use an opening tag of <ol> followed by a nested <li> item.
Each <li> is closed with </li> The final item is closed and then has the closing tag of the ordered list </ol>

Ordered lists have a number of attributes which can be applied.

  •  reversed  Reverses number order.
  •  start="123"  Choose a number to start the list from.
  •  type="A"  Set the type of numbering. This is used for legal docs and docs where the numbering system is used for referencing. Can be applied to individual list elements or the entire list.
    •  a Lowercase letters.
    •  A Uppercase letters.
    •  i Lowercase roman numerals.
    •  I Uppercase roman numerals.
    •  1 Numbers (default).

Unordered Lists

  • Items in a list.
  • Items will be bullet pointed.
  • Used for things which the order is not important.
  • An ingredients list for example.

Unordered lists use an opening tag of <ul> followed by a nested <li> item.
Each <li> is closed with </li> The final item is closed and then has the closing tag of the ordered list </ul>
Bullet point style can be styled using the css list-style-type property.

Nested Lists

Nests can be listed inside each other. There is no rule on unordered and ordered. See code above for basic list nesting.
