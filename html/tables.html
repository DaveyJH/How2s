Tables

Tables are used for data. Consisting of a table <table>, table rows <tr>, table headings <th> and table data <td>.
Each element requires a closing tag.

The format of a table is as follows:

<table>
<tr> </tr>
<th>Heading 1</th>
<th>Heading 2</th>
<th>Heading 3</th>
<tr> </tr>
<td>1st Data - H1</td>
<td>1st Data - H2</td>
<td>1st Data - H3</td>
<tr> </tr>
<td>2nd Data - H1</td>
<td>2nd Data - H2</td>
<td>2nd Data - H3</td>
</table>

See code for simple example

Heading 1 Heading 2 Heading 3
Data1Heading1 Data1Heading2 Data1Heading3
